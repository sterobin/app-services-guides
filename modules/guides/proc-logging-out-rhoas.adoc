= Logging out of rhoas
:imagesdir: ../_images

[role="_abstract"]
You can log out from the `rhoas` CLI by using the `rhoas logout` command.

.Procedure

* Log out of `rhoas`.
+
[source,shell]
-----
$ rhoas logout
Successfully logged out
-----
